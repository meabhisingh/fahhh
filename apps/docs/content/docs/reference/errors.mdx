title: Errors

`fahhh` provides a base `HttpException` class and a suite of predefined exception classes for standard 4xx and 5xx HTTP status codes.

You can import these exceptions from `fahhh/errors` and throw them from your route handlers. `fahhh` will automatically catch them and send a properly formatted JSON response to the client.

## `HttpException`

The base error class used for all HTTP exceptions.

```typescript
export interface HttpExceptionOptions {
  headers?: HeadersInit;
  cause?: unknown;
  body?: unknown;
}

export class HttpException extends Error {
  readonly status: number;
  readonly statusText: string;
  readonly headers?: HeadersInit;
  readonly body?: unknown;

  constructor(
    status: number,
    message: string,
    statusText = message,
    options: HttpExceptionOptions = {},
  );
  
  toResponse(): Response;
}
```
