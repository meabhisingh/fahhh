## `src/api/`
This directory holds your backend code. Any file exported from here becomes an API route accessible by the frontend. For example, `src/api/users/index.ts` becomes available at the `/api/users` route.

## `src/web/`
This directory contains your React application. Your entry points, React components, and client-side logic live here. You can import your backend endpoints directly from `virtual:api`.

## `fahhh.config.ts`
The configuration file for `fahhh`. Here you define the locations of your API and web directories, as well as the port for your API server.

```typescript title="fahhh.config.ts"
import { defineConfig } from "fahhh";

export default defineConfig({
	apiDir: "src/api",
	webDir: "src/web",
	apiPort: 8787,
});
```

## `vite.config.ts`
Your standard Vite configuration file. By default, it comes configured with the React plugin, React Compiler, and Tailwind CSS v4.
